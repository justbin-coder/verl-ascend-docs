## 📖 关于本项目

欢迎来到昇腾VERL的官方文档中心！

本项目是 **VERL** 训练框架在 **昇腾（Ascend）** AI 平台的官方文档资源库。我们致力于为全球开发者提供最全面、最精准、最易于上手的学习资源，帮助您在昇腾硬件上高效地使用 VERL 进行 AI 模型训练。

本文档遵循 **"Docs-as-Code"** 的理念，将所有文档源文件在GitCode上进行版本管理，并通过CI/CD流水线自动构建和发布。这不仅确保了文档与代码的同步更新，也使社区贡献变得前所未有的简单。

## 🤝 如何贡献

我们热烈欢迎并鼓励社区开发者共同参与本文档的建设，任何形式的贡献都是对社区的宝贵价值！无论是修正一个错别字，优化一句表达，还是撰写一篇全新的教程，我们都无任欢迎。

贡献流程非常简单：

1.  **发现问题或有新想法？** 在本仓库的 [**Issues**](https://gitcode.com/your-username/verl-ascend-docs/issues) 页面提出您的建议。
2.  **Fork 本仓库** 到您自己的账户下。
3.  **创建新的分支** (e.g., `feature/add-resnet-guide` or `fix/typo-in-quickstart`)。
4.  在您的分支上进行修改和内容撰写。
5.  **提交一个 Pull Request (PR)** 到本仓库的 `main` 分支，并清晰描述您的修改内容。

我们会尽快Review您的PR，一经合并，您的贡献将自动发布到官方文档网站，并出现在贡献者列表中！
